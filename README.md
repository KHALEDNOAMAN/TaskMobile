## ✨ Features

### ✅ Task Management
- Create tasks with title, description, priority (High/Medium/Low), and due date
- Swipe-to-complete and swipe-to-delete gestures
- Filter by: All, Active, Completed, Overdue
- Sort by priority, due date, or creation date
- Category tags with color coding

### 🔥 Habit Tracking
- Daily habit checklist with tap-to-complete
- Streak counter with fire emoji animation
- Weekly completion heatmap (GitHub-style)
- Habit categories: Health, Productivity, Learning, Fitness, Mindfulness
- Best streak record tracking

### 📊 Analytics Dashboard
- Weekly task completion rate (bar chart)
- Habit streak leaderboard
- Productivity score trend
- Category breakdown (pie chart)
- Daily/Weekly/Monthly view toggle

### 🏆 Achievements & Gamification
- Achievement badges for milestones (7-day streak, 100 tasks, etc.)
- Level system based on completed tasks and streaks
- Progress bar showing XP to next level
- Motivational quotes on home screen

## 🚀 Getting Started

```bash
git clone https://github.com/KHALEDNOAMAN/TaskMobile.git
cd TaskMobile
npm install
npx expo start
```

Then scan the QR code with **Expo Go** app on your phone, or press `w` for web preview.
